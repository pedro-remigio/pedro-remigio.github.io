# Remigio Eventos e Serviços

Projeto desenvolvido para o Check-point de Front-end da faculdade.

## Objetivo

Demonstrar conhecimentos em:

- HTML semântico;
- Tailwind CSS;
- Responsividade;
- JavaScript;
- DOM e eventos;
- Validação de formulário;
- Publicação estática.

## Funcionalidades

- Menu responsivo mobile;
- Renderização dinâmica de eventos;
- Filtro de eventos;
- Formulário com feedback visual;
- Layout responsivo;
- Navegação entre páginas.

## Tecnologias utilizadas

- HTML5
- Tailwind CSS
- JavaScript Vanilla

## Publicação

Projeto publicado via GitHub Pages.
